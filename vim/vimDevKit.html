Overview

Vim is a clone, with additions, of Bill Joy's vi text editor program for Unix. Vim's author, Bram Moolenaar, based it upon the source code for a port of the Stevie editor to the Amiga and released a version to the public in 1991.

Plugins
PavedRoad has installed 5 vim plugins. As a Developer the two plugins you need to be aware of are vim-go and deoplete-go. These supply respectively the vim go commands and vim go completion. The other three plugins are pre-requites for the deoplete-go plugin:
  • deoplete-go requires:
      • deoplete.nvim plugin
  • deoplete.nvim plugin requires:
      nvim-yarp plugin
      vim-hug-neovim-rpc plugin
For now we rely on the default settings for the plugins. More information about the vim-go settings can be located in section 7 of its document page. All of the settings for deoplete-go are described in the Available Settings section of its main landing page.

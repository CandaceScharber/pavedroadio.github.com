PavedRoad DevKit Download

PavedRoad DevKits provides the tools to install a cloud native software development environment. The development environment supports versions of the bash shell and the vim editor that have been enhanced through completions and plugins to supprt a cloud native environment. In addition the project installs the go language and relevant packages along with the git command and other minimally needed development software. Jump to download

Pre Requisites

Dependency Description
Docker

Docker is a platform which packages an application and all its dependencies together in the form of containers. This containerization aspect of Docker ensures that the application works in any environment. Kubernetes is a platform for automating deployment, scaling, and operating application containers across clusters of hosts, so the first order of business to use Kubernetes is to create containers for your applications.

Virtual Box

A virtual machine (VM) is a software program or operating system that not only exhibits the behavior of a separate computer, but is also capable of performing tasks such as running applications and programs like a separate computer. Kubernetes can run on VMs or Bare Metal, but for local development it is typically more efficient to use VMs.

Go-Vim DevKit Contents

Tool Description

Go

Go is an open source programming language from Google, and has become the defacto choice for developing Cloud Native Microservices. Many Go packages are available either as development tools or as base libraries. Some of these Go tools may have been developed as bash completions or vim plugins. PavedRoad makes it easy by pre-integrating Go package tools as well as completions and plugins. Click here for a complete list.

Vim

Vim is a highly configurable text editor for efficiently creating and changing any kind of text. PavedRoad has enhanced Vim to be a complete IDE adding word completion and development related shell commands that can be run inside of Vim. These commands are mapped to keystrokes for execution allowing developers to work without ever having to leave Vim. If the version of Vim on the target device is not 8.0 or later then the latest version of will be installed. Click here for more information.

Skaffold

Skaffold is a command line tool that facilitates continuous development for Kubernetes applications. You can iterate on your application source code locally then deploy to local or remote Kubernetes clusters. Skaffold handles the workflow for building, pushing and deploying your application. Click here for more information.

Enhanced Bash

The bash shell, aliases, and completions are integral parts of any development environment. PavedRoad has included a list of helpful bash-completions. Command completion is especially important in a development environment as there are commands like git that have many sub commands and options. In addition to completion capabilities, two sets of aliases are installed, one set for GoLang and the other for git commands. Click here for a complete list.

Kompose

Kompose is a tool to help users familiar with docker-compose move to Kubernetes. It takes a Docker Compose file and translates it into Kubernetes resources. Click here for more information.

Download PavedRoad DevKit

Download the bootstrap script for your target OS and run it to install the DevKit. This script installs curl, git, and salt on the target device and then clones the PavedRoad DevKit repository. SaltStack (in masterless mode) is used to install the development environment. The bootstrap script runs salt-call to apply the salt states in the cloned repository on the target device. Running the bootstrap script installs the complete development environment, no knowledge about SaltStack is required. Copy the curl command for your target OS and paste into your systems terminal shell.

Operating System Download Command

Linux

curl -L https://raw.githubusercontent.com/pavedroad-io/pavedroad/master/devkit/bootstrap-unix.sh | /bin/bash

curl -L https://raw.githubusercontent.com/pavedroad-io/pavedroad/master/devkit/bootstrap-macos.sh | /bin/bash

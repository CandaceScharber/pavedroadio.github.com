Skaffold Quickstarts and Training

Overivew

Skaffold is a relatively new tool put into the Open Source community by Google. It is a command line tool that manages the workflow for building, pushing and deploying application updates to Kubernetes. One distinguishing feature is that it is designed to allow developers to continue to iterate code locally, then build and deploy the updates to either a local or a remote Kubernetes cluster. This can make a huge difference in productivity of a developer, especially in the early stages of development by shortening the time it takes between each iteration. You can iterate on your application source code locally then deploy to local or remote Kubernetes clusters. Skaffold handles the workflow for building, pushing and deploying your application.

Skaffold can also be used in an automated context such as a CI/CD pipeline to leverage the same workflow and tooling when moving applications to production.Skaffold is highly configurable and supports Helm for deployment. Skaffold does not have a server-side component, and it can be safely used in non-interactive CI/CD systems.
